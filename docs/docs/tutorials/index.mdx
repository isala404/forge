## What You'll Learn

By completing these tutorials, you'll understand:

- **Schema Design** - How to model your data in Rust
- **Queries & Mutations** - How to build your API
- **Real-Time** - How subscriptions work
- **Background Jobs** - How to process work asynchronously
- **Authentication** - How to protect your app
- **Frontend Integration** - How to use FORGE with Svelte

## Prerequisites

Before starting, make sure you've completed the [Quick Start](/quick-start) guide and have:

- A working FORGE installation
- Basic Rust knowledge (don't worry, we'll explain the patterns)
- Basic TypeScript/JavaScript knowledge
